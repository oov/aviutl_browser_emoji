# 絵文字.obj

@git_tag@ ( @git_revision@ )

[aviutl_browser](https://github.com/oov/aviutl_browser/) で絵文字を表示するカスタムオブジェクトです。

# インストール

絵文字.obj は aviutl_browser に標準で付属しているので、別途インストールは必要ありません。

個別にバージョンアップする場合は exedit.auf と同じ場所にある script フォルダーを上書きしてください。

# 使い方

カスタムオブジェクトで「絵文字」を選ぶと使用できます。

「絵文字セレクター」フォルダーにある HTML ファイルをブラウザーで開くと、絵文字一覧が表示されます。  
絵文字をクリックすると「絵文字コード」がコピーでき、それをカスタムオブジェクトの設定に貼り付けると自由な絵文字が利用可能です。
